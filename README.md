## Cinnamon - Applet 路径
/usr/share/cinnamon/applets  
~/.local/share/cinnamon/applets

## 重载 applet
任务栏右键 - 疑难解决 - 重启 Cinnamon

## Cinnamon Applet 调试
打开命令窗口：Alt + F2，输入 lg，Log 标签查看错误。
